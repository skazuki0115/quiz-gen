ブラウザだけで「AIが自動で四択クイズを作ってくれる」Next.jsアプリです。
お題を入力するだけの「練習モード」、PDFを読ませて問題を作る「PDFモード」、連続出題する「テストモード」があります。
作った問題はアプリ内の履歴ページで振り返りできます。
できること

好きなテーマで即席クイズを作る
PDF（5MBまで）をアップロードし、その内容からクイズを作る
練習モード：1問ずつ出題 → 解説表示
テストモード：まとめて出題・採点
履歴ページ：作った問題一覧とPDFから使われた範囲のカバレッジを確認
使い始める準備

Node.js 18 以上をインストール
依存パッケージを入れる: npm install
環境変数を設定
.env.local を作成し OPENAI_API_KEY=あなたのAPIキー を書く
任意でアンケートリンクを出したい場合 NEXT_PUBLIC_SURVEY_FORM_URL=<URL>
開発サーバー起動: npm run dev（デフォルト http://localhost:3000）
画面の使い方（ブラウザ）

トップページでモードを選択
練習モード：テーマ入力 → 難易度選択 → 「クイズを作る」
PDFモード：PDFをドラッグ&ドロップor選択 → （必要ならテーマ入力）→ 「PDFからクイズを作る」
テストモード：名前・テーマ・難易度を入れて開始
右上のスピーカーボタンで効果音ON/OFF
「履歴」でこれまでの問題と解説を確認
仕組み（ざっくり）

generate-quiz.js: OpenAI APIに指示を送りクイズを生成
upload-pdf.js: PDFをテキスト化し、ベクトルDB（SQLite）に分割・保存
quizGenerator.js: 出題指示文の整形・回答シャッフルなど
vectorStore.js / history.js: SQLiteでPDFのチャンクとクイズ履歴を管理（データは quiz-history.db に保存）
保存場所とプライバシー

作った問題・PDFの分割データはローカルの quiz-history.db に保存されます。
OpenAI APIに送る内容（入力テキストやPDFの分割テキスト）には機密情報を入れないでください。
トラブルシュート

クイズが作れない：OPENAI_API_KEY が正しいか確認
PDFアップロード失敗：5MB超過やパスワード付きPDFでないか確認
文字化けしている：ファイルの文字コードを UTF-8 にして再保存
もう一度やり直したい：画面の「戻る」またはトップに戻って再入力
開発者メモ（必要なら）

主要コマンド: npm run dev / npm run build / npm start
データベースはBetter SQLite 3、ベクトル検索は単純なドット積ランキング。
UIコンポーネントは components/、スタイルは styles/ に配置。
